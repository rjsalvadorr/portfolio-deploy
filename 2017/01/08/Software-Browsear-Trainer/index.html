Browsear Trainer

Jan 2017
See the full app here!

Around Fall/Winter 2016, I really wanted to be a better musician. That involved training aural skills. Normally, that takes a long time to develop, but I was convinced that software can make that a bit faster.

Heads up! This post contains a lot of jargon related to music theory. Also, the app plays music through the browser. And was only tested for Chrome and Firefox on desktop

This all started when I first saw this video describing an ear training exercise:

  1. Have a pianist buddy play a cadence to confirm the sound of a particular key in your ear.
  2. Piano bud then plays a note in that key
  3. You have to identify that note/scale degree just by listening to it

By all accounts, it’s an effective exercise. After discovering this method, the first thing I thought was: “Hey, I can write that. It’s probably going to be easy too…“

Game plan

I wanted the app to do the following on a browser:

  1. Play a cadence to confirm the tonic in users’ ears.
  2. Play the test note.
  3. Ask the user to guess the interval.
  4. If the user got it right, reset the quiz.
    4b. If the user got it wrong, give them the option to replay the sounds, and guess until they get it right.
  5. Rinse and repeat, until the user is really good at playing by ear.

Bonus objectives:

  • Allow ear training in a minor key.
  • Melodic dictation. This involves asking the user for the specific scale degrees used in a melody.

Pulling it off

For this project, I decided to use the following tools/libraries:

  • teoria, a JavaScript library for music theory.
    This allowed me to use the correct chords for random keys, and send the right MIDI keys to the midi player.
  • midi-player-js and midi-writer-js, JavaScript libraries for writing and playing MIDI notes on a browser. This is one component I used to play MIDI notes through the browser.
  • soundfont-player, another JS library. This is what did most of the heavy lifting for actually playing sounds.

Results

I hit just about all my targets. I could use this tool ad nauseam for both major and minor keys. I noticed a bit of improvement in my scores as I kept using this thing. So something was improving. I felt like my listening skills improved, but I can’t really verify if this tool made a huge impact for me.

Would I recommend this tool for others? Probably not. I’ve found better methods for ear training. My recommendation for those interested: TRANSCRIBE LOTS OF MUSIC and PLAY ALONG TO RECORDINGS. Those activities will force you to listen actively, and they’re now my favourite ways to train my ears.

All in all, getting MIDI notes to play on a browser was tons of fun. I could probably take a lot of this code and make some more toys for computer-aided composition. Or maybe some browser-based instrument?
