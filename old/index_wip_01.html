Portfolio

Hotel Website

A website for a hotel, which lets clients search for available rooms and make reservations. Implemented as a Java web app, using an Oracle 11g database.

Hairstylist Scheduler

Website for a hairstylist, showing information about the business. Also allows clients to schedule appointments. Uses PHP and MySQL in the back-end.

FlowFlix

A movie database, emulating a video rental service. Allows users to 'rent' movies, and lets administrators add new movies to the database. Written in ASP, using a SQL Server database.

BroQuest (working title)

A game written in C++, using the SDL library. During the summer of 2013, my brother and I worked on this 2D game for PC. He handled the artwork and design, while I handled the programming.
