Fretboard quizzer

This is for a little quiz for learning the natural notes on the guitar fretboard! All done in HTML/Javascript/CSS. If you're viewing this on a mobile device, the author recommends doing it in landscape orientation.

Instructions

  • Find the locations of the note that's shown.
  • To mark a location, click the string in the correct fret position
  • If it's an open string, click on the nut.
